In this project I used a dataset from a PSID Family Income and Wealth data set that I got from the PSID website for this project as part of my instructions. This was for the class DAT402, and I applied pandas for data analysis and Scikitlearn for machine learning. The first thing I did was quickly analyze my data and fix any issues with the data. After this, I added categorical lables to the dataset for machine learning models that use multiple regeression. After performing machine learning on this dataset I then used data visualizations to draw conclusions about the data. I did the same for other types of machine learning such as deterministic categories.
